JSAPI.info

JSAPI.info is a JavaScript library/framework source inspector. It was made with jQuery in mind, but will work with other libraries. Libs are added/configured in libs.json.

Start exploring:

  • jQuery
  • MooTools
  • Dojo
  • Underscore
  • YUI
